Harmonograph User Manual

Rotary Harmonograph Simulator for Pen Plotter Art · Browser-based · SVG Export · Multi-pass
Version 1.0.0 · March 2026 · Nate Northrup

1. Introduction

Harmonograph is a browser-based simulator of a rotary (Lissajous) harmonograph — a mechanical device that traces beautiful curves using the interaction of oscillating pendulums. It is designed for artists who work with pen plotters such as the AxiDraw, and produces high-fidelity SVG files ready for plotting on any paper size.

Everything runs in a single HTML file with no installation required. Open it in any modern browser and start creating immediately.

What is a Harmonograph?

A physical harmonograph suspends a pen from two or more pendulums swinging at right angles. As the pendulums slowly lose energy, the pen traces an evolving Lissajous curve onto paper. The resulting patterns are determined by the frequency ratios, phases, and decay rates of the pendulums.

This simulator faithfully replicates that mathematics, adding controls not possible with physical hardware: multiple independent pen passes, exact parameter tuning, instant randomisation, and unlimited experimentation with undo/redo support.

Key Features

  • Multi-pass system — layer multiple independent pen paths, each with its own colour, shape, and transform settings
  • 25 frequency ratio combinations per axis for precise Lissajous control
  • Phase multiplier for petal and star rosette patterns
  • Detune and asymmetric detune for slowly rotating and spiralling forms
  • Amplitude decay for enveloped curves that collapse to a point
  • Full transform controls: phase offset, X/Y offset, scale, and rotation
  • Per-pass randomisation with lockable parameters
  • 8 built-in presets plus unlimited user presets with JSON export/import
  • 40-state undo/redo history
  • SVG export with per-layer group elements in real millimetre coordinates
  • All numeric values are directly editable by clicking on them

Getting Started

Open index.html (or visit the hosted version) in any modern browser — Chrome, Firefox, Edge, or Safari. The canvas appears on the right and the control panel on the left. A default curve loads immediately.

Tip: Try clicking one of the eight built-in presets in the Presets panel at the bottom of the sidebar to see different starting points before you begin tweaking.

2. Interface Overview

The interface is divided into two areas: the sidebar on the left containing all controls, and the canvas on the right showing the live preview. Every parameter change redraws the canvas in real time.

Sidebar Layout

The sidebar is organised into four stacked panels:

  • Top Panel — Paper size selector, seed input, undo/redo buttons, and the SVG export button
  • Pens Panel — One or more pen pass cards, each containing all per-pass controls. An “Add Pass” button creates additional passes
  • Global Panel — Background colour, density, line weight, and opacity — settings that apply to all passes
  • Presets Panel — Eight built-in presets and your saved user presets

Canvas

The canvas updates automatically whenever any parameter changes. A thin progress bar at the bottom of the canvas appears during longer redraws (high density settings). The canvas always matches the selected paper size and orientation.

Editable Values

Every numeric readout in the interface can be clicked to type a value directly. Editable fields show the current value in the appropriate unit (mm for line weight, rad for phase values, etc.). Press Enter or click away to apply. The canvas redraws immediately.

Tip: You can type values outside the slider range in the text field — the slider will clamp, but the typed value will be applied.

4. Pen Passes

A pen pass represents one layer of the drawing — one colour of ink, one pen, one run of the harmonograph. You can have as many passes as you like, each with completely independent shape, colour, and transform settings.

Pass Header

At the top of each pass card is a header bar with the following controls:

  • Drag Handle — Grab and drag this icon (the dotted grid symbol at the left) to reorder passes. You must drag from this icon specifically — dragging elsewhere on the card moves the scrollbar.
  • Colour Swatch — Click to open the colour picker and choose this pass’s stroke colour.
  • Visibility — Toggle this pass on or off. Hidden passes still exist and can be re-enabled at any time.
  • Solo — Show only this pass and hide all others. Click again to exit solo mode.
  • Duplicate — Create an exact copy of this pass, appended below it.
  • RND — Randomise all unlocked parameters for this pass using the current seed.
  • Remove — Delete this pass. Not shown when only one pass exists.

Lock Row

Directly below the header is a row of lock buttons. Locked parameters are excluded from randomisation. This lets you fix elements you like (such as the colour or frequency ratio) while freely randomising others.

Col RX RY Amp Ph Det
  • Col — Lock the pass colour
  • RX — Lock the X frequency ratio
  • RY — Lock the Y frequency ratio
  • Amp — Lock oscillator amplitudes A1–A4 and phases p1–p4
  • Ph — Lock the phase offset
  • Det — Lock detune X and detune Y

Adding and Managing Passes

  • Click Add Pass at the top of the Pens panel to add a new pass with default settings.
  • Click Randomise All (also in the Pens panel header) to randomise all passes at once, respecting their individual locks.
  • Use the drag handle to reorder passes — order affects visual layering (later passes draw on top).
Tip: Solo mode is very useful when tweaking one pass in a complex multi-pass composition — you can focus on it without visual interference from other layers.

5. Shape Controls

The shape section controls the mathematical form of the harmonograph curve for each pass. These are the most powerful parameters in the tool.

Frequency Ratios

Two ratio grids — one for the X axis and one for the Y axis — each let you select from 25 musical frequency ratios. The ratio determines the fundamental shape of the Lissajous figure.

Available Ratios

1:12:13:14:15:16:17:18:1 3:25:27:2 4:35:37:38:3 5:47:4 6:57:58:5 7:68:7

Simple ratios (1:1, 2:1, 3:2) produce clean, closed figures. Complex ratios produce intricate lattice patterns. Different X and Y ratios create asymmetric forms.

Tip: For pen plotting, start with 3:2 or 5:4 on both axes — these produce stable, elegant curves. Higher prime ratios like 7:5 create beautiful but denser lattice patterns.

Detune

Detune adds a tiny frequency drift to the oscillators, causing the Lissajous figure to slowly precess — rotate or spiral over time rather than remaining stationary. This is what makes harmonograph curves so organically complex.

ControlRangeDescription
Detune X0.000–0.0125Frequency drift applied to both X-axis oscillators. Even tiny values create visible precession
Asym YtoggleEnables independent Y-axis detune. When off, Y uses the same detune as X
Detune Y0.000–0.0125Visible only when Asym Y is enabled. Sets independent drift on the Y oscillators
Tip: Matching detune X and Y creates a spinning rosette. Different X and Y values create a slow precession that produces entirely different character in the curve.

Amplitude Decay

Decay simulates the real physics of pendulums gradually losing energy. As the curve progresses, the amplitude on each axis decreases exponentially, causing the figure to spiral inward.

ControlRangeDescription
Decay X0–10Exponential amplitude decay on the X axis. 0 = no decay (constant amplitude)
Decay Y0–10Exponential amplitude decay on the Y axis. Independent from X
Note: Decay of 0 means the curve never collapses — it continues at constant amplitude for the full density. At a decay of 10, the curve shrinks to nearly a point by halfway through the draw.
Tip: Asymmetric decay (e.g., X=2, Y=0) creates curves that collapse to a horizontal or vertical line instead of a central point, yielding distinctive elongated spirals.

Oscillator Tabs (P1–P4)

Each pass uses four oscillators: P1 and P2 combine to drive the X axis, while P3 and P4 drive the Y axis. Tabs P1–P4 give access to each oscillator’s individual amplitude and phase.

ControlRangeDescription
A1–A4 (Amplitude)0.00–1.00The contribution of each oscillator. A1+A2 are automatically normalised so their sum ≤ 1; likewise A3+A4
p1–p4 (Phase)0.00–6.28 radStarting phase of each oscillator in radians (0–2π). Shifting the relative phase changes the curve shape
Tip: For clean ellipses, keep A2=0 and A4=0. To add complexity without changing the fundamental ratio, gradually increase A2 or A4 and adjust their phases.

6. Transform Controls

Transform controls adjust the position, size, and orientation of a pass on the canvas without affecting the mathematical shape of the curve.

ControlRangeDescription
Phase Offset0.00–6.28 radRotates the starting phase of the entire Lissajous figure. Useful for aligning multi-pass compositions
X Offset−1.00 to 1.00Translates the curve horizontally. 0 = centred. ±1.0 = full canvas width
Y Offset−1.00 to 1.00Translates the curve vertically. 0 = centred
Scale0.10–2.00Scales the curve. 1.0 = full canvas size. Values above 1.0 allow curves to extend beyond the margin
Rotation−180° to 180°Rotates the rendered curve around the canvas centre

A Reset Transform button appears at the bottom of the transform section to restore all five values to their defaults (phase offset 0, offsets 0, scale 1, rotation 0).

Tip: To layer two passes offset from centre, set X Offset to +0.15 on one and −0.15 on the other. Combined with different colours, this creates an overlapping composition that plots in two pen passes on the AxiDraw.

7. Phases

The phases multiplier repeats the base Lissajous figure multiple times, with the phase offset incremented evenly between repetitions. This creates symmetric petal, star, and rosette patterns from a single pass.

1× (no repetition), 2×, 3×, 4×, 6×, 8×

At 1×, a single Lissajous curve is drawn. At 6×, six copies of the curve are drawn, each rotated by 60° of phase offset, producing a six-petalled rosette.

Tip: Phases work best with symmetric ratio combinations like 1:1, 3:2, or 5:4. With 4× phases and a 3:2 ratio, you get a clean four-petalled flower. With 6× and 5:4 you get an intricate 24-lobe star.
Note: The density setting applies to the total curve — at 6× phases and 30,000 density, each repetition gets 5,000 segments. Increase density when using high phase counts.

8. Randomisation & Locking

Harmonograph is designed for exploratory use. The randomisation system lets you generate thousands of unique curves quickly while preserving the elements you want to keep.

Per-Pass RND Button

The RND button on each pass card randomises all unlocked parameters for that pass. The current global seed determines the randomisation sequence, so the same seed always produces the same random result.

Randomise All

The Randomise All button (in the Pens panel header, above the first pass card) applies RND to every pass simultaneously. Each pass respects its own lock state.

Seed Control

The seed is a global integer. Use the arrow buttons to step through seeds sequentially, or the dice button to jump to a random seed. The value field is directly editable.

Tip: A useful workflow: lock the parameters you like (e.g., colour and ratios), then cycle through seeds with the arrow keys until you find a pleasing composition. Because locked parameters are preserved, you’re essentially exploring a constrained space of variations.

Parameter Locks

Each pass has six independent locks in the lock row. A locked parameter is shown as highlighted and is completely ignored by the RND button. Locks persist when you duplicate a pass or load a preset.

  • Col — Protect the colour you’ve chosen
  • RX / RY — Protect the X and Y frequency ratios independently
  • Amp — Protect all oscillator amplitudes and phases
  • Ph — Protect the phase offset
  • Det — Protect the detune X and detune Y values

9. Presets

Presets save and restore the complete state of all passes and global settings. The Presets panel is at the bottom of the sidebar.

Built-in Presets

Eight built-in presets demonstrate the range of forms the tool can produce:

  • Lobed — Smooth three-lobed rosette
  • Trefoil — Three-fold symmetric star with decay
  • Star — Sharp angular star with phase multiplier
  • Spiral — Open spiral using high detune
  • Ribbon — Twisted ribbon with asymmetric detune
  • Lattice — Dense lattice from complex ratio combinations
  • Rose — Multi-petal rose using phases multiplier
  • Figure-8 — Classic lemniscate with decay
Note: Built-in presets cannot be deleted or overwritten. Clicking a built-in preset loads it immediately — your current state is pushed onto the undo stack first, so you can always undo back.

User Presets

To save your current state as a user preset, click the Save button in the Presets panel and enter a name. User presets appear in a separate list below the built-ins.

To load a user preset, click its name. To delete it, click the × button next to it.

Export and Import

Click Export Presets to download all user presets as a JSON file. Click Import Presets to load presets from a JSON file — useful for sharing presets with others or backing them up.

Note: User presets are stored in browser localStorage. Clearing browser data will delete your presets. Use Export Presets regularly to keep backups.
Tip: To share a preset with another user, export your presets JSON, send the file, and they import it on their end. All pass parameters, colours, and settings are preserved exactly.

10. Undo & Redo

Harmonograph maintains a 40-state undo history. Every parameter change, randomisation, preset load, pass addition or deletion, and reorder pushes a snapshot onto the undo stack.

ShortcutAction
Ctrl+ZStep back to the previous state
Ctrl+YStep forward to the next state (after an undo)
Ctrl+Shift+ZRedo (alternative shortcut)

The Undo and Redo buttons in the top panel are greyed out when there is nothing to undo or redo. The history is cleared when you refresh the page.

Tip: Before a big randomisation session, save your current state as a user preset first. The undo stack is 40 steps deep — enough for exploratory use, but a named preset gives you a permanent save point.

11. SVG Export

Click the Export SVG button in the top panel to download the current drawing as a scalable vector graphic (.svg file).

File Format

The exported SVG:

  • Uses real millimetre coordinates matching the selected paper size
  • Contains one <g id="layer0">, <g id="layer1">, etc. group element per pass
  • Each group’s stroke colour matches the pass colour
  • Stroke width is set to the global line weight in mm
  • Each line segment is an individual <line> element

Pen Plotter Workflow

For pen plotters such as the AxiDraw, the recommended workflow is:

  1. Set paper size to match your physical paper
  2. Set line weight to match your pen nib width
  3. Export SVG
  4. Open in Inkscape or your plotter’s control software
  5. Each SVG layer (group) corresponds to one pen — plot layer by layer, swapping pens between
Note: The SVG file can be opened directly in a web browser for inspection. Zoom in to check detail before plotting.
Tip: For fine nibs (0.2–0.3 mm), a density of 20,000–30,000 produces a smooth result. For brush pens or thicker nibs, 5,000–10,000 is often sufficient and plots faster.

13. Tips & Techniques

Creating Multi-Pass Compositions

The most compelling results often come from two or three passes with complementary colours. A useful starting approach:

  • Set up one pass with a cool colour (blue, teal) and a simple ratio like 3:2
  • Duplicate it and change the colour to a warm tone (orange, red)
  • On the second pass, set a different detune value to create divergence between them
  • Use visibility toggles to compare passes individually before previewing the full composition

Controlling Density vs. Complexity

Density and frequency ratio interact strongly. A simple 1:1 ratio with 50,000 lines will overdraw the same path into a solid band. A complex 7:5 ratio needs more lines to fill in the lattice gaps. As a rule:

  • 1:1 and 2:1 ratios: 5,000–15,000 lines
  • 3:2 and 5:3 ratios: 10,000–25,000 lines
  • 7:5, 8:7, and other complex ratios: 20,000–50,000 lines

Using Decay Effectively

Amplitude decay is one of the most characterful controls. Some patterns to explore:

  • Equal X and Y decay (e.g., both at 2.0): smooth inward spiral ending at a central point
  • High X decay, zero Y decay: the curve collapses to a horizontal line
  • Zero decay with high detune: the curve precesses but never collapses, filling in a torus-like band
  • Low decay (0.2–0.5): subtle envelope that adds just a hint of natural physics

Seed Exploration Strategy

The most efficient way to explore is:

  1. Choose a ratio combination and phase count you like
  2. Lock RX and RY to preserve them
  3. Press RND repeatedly (or step through seeds) to vary the phases and amplitudes
  4. When you find something promising, save it as a user preset before continuing

Preparing for the Plotter

Before exporting for plotting, verify:

  • Paper size matches the physical paper you’ll use
  • Line weight is appropriate for the pen nib
  • Opacity is 255 (fully opaque) — the plotter doesn’t interpret transparency
  • Each pass uses a distinct colour that corresponds to an actual pen you have

14. Mathematical Reference

For those interested in the underlying physics, the harmonograph equations that power each frame are:

Coordinate Equations

x(t) = A1 · cos(f1 · t + p1 + φ) + A2 · cos(f2 · t + p2)
y(t) = A3 · cos(f3 · t + p3 + φ) + A4 · cos(f4 · t + p4)

Variable Definitions

SymbolMeaningDescription
f1, f2X frequenciesX-axis frequencies, scaled by (1 + detuneX)
f3, f4Y frequenciesY-axis frequencies, scaled by (1 + detuneY)
φ (phi)Phase offsetThe per-pass phase offset control (0–2π rad)
A1–A4AmplitudesOscillator amplitudes; A1+A2 and A3+A4 each normalised to ≤ 1
p1–p4PhasesStarting phase of each oscillator (0–2π rad)
tTimeRuns from 0 to totalT = density × dt (dt = 0.04)

Amplitude Decay

If decay is non-zero, the amplitudes are multiplied by an exponential envelope at each time step:

A(t) = A · exp(−decayX · t / totalT)

This is applied independently to the X and Y amplitudes using decayX and decayY respectively.

Phase Multiplier

When phases > 1, the curve is drawn N times. On each repetition k (k = 0 to N−1), the phase offset is incremented by:

φ_k = φ_0 + k × (2π / N)

This evenly distributes N copies of the figure around the phase circle, producing symmetric N-fold patterns.
